# shit-bin
💩 bin's and config for shit defi activity

---

develop and publish this package

```bash
bun install
bun update --latest
bun run tsc --noEmit
# bun run index.ts
bun run build
bun publish --dry-run
bunx npm login
bun publish --access public

# prettier
bunx prettier . --write
bunx prettier . --check
```

---

## HOW TO USE

add

```sh
npm i @huggies/shit-bin
bun add @huggies/shit-bin
```
